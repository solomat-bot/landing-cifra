## 1. Концепция проекта

### 1.1 Миссия
Помогать микробизнесу наводить порядок — и в процессах, и в деньгах. Одна команда закрывает оба направления: финансы (Евгения — управленческий учёт, финмодели, отчётность) и технологии (Ольга — автоматизация сбора данных, сайты, боты, AI).

### 1.2 Позиционирование (с 2026-06-17)
**Финансы — ядро, технологии — усиление.**
- Евгения — основной драйвер: 5+ лет опыта, работа с маркетплейсами и онлайн-школами, recurring-доход 30–75k/мес
- Ольга — цифровая инфраструктура: автоматизация сбора данных, сайты, боты, AI
- Вместе: единая система + технологическая обвязка

### 1.3 Формат
Одностраничный лендинг (14 секций) с презентацией команды, услуг и цен.
- **Главная:** финансы как основа + технологии как отдельный блок
- **Калькулятор:** 3-шаговая диагностика с ползунком (время → боль → ситуация → результат)
- **ЦА:** маркетплейсы (WB, Ozon), онлайн-школы, сервисный бизнес

### 1.3 Ключевые принципы
- **Двойная экспертиза:** технологии + финансы под одной крышей
- AI — помощник, а не замена человеку
- Ультра-бюджетные цены на старте (набор портфолио)
- Честность и прозрачность: фиксированные цены, реальные сроки

## 2. Рынок и тренды 2025-2026

### 2.1 Тренды в разработке лендингов
- **Минимализм и скорость:** чистый дизайн, оптимизация под Core Web Vitals
- **Микроанимации:** subtle animations для вовлечения (GSAP, Framer Motion)
- **AI-персонализация:** динамический контент под сегмент посетителя
- **Темные темы:** поддержка dark mode как стандарт
- **Mobile-first:** более 70% трафика с мобильных устройств
- **Интерактивные элементы:** калькуляторы, квизы, 3D-элементы

### 2.2 Тренды в Telegram-ботах
- **Telegram Mini Apps (TMA):** конверсия в 2-3x выше, чем при отправке на сайт
- **AI-боты:** интеграция ChatGPT/Smartbot AI для естественного общения
- **Геймификация:** уровни, ежедневные награды, лидерборды для LTV
- **Сквозная аналитика:** бот + CRM + платежи в одной экосистеме
- **Автоворонки:** лид-магнит → трипваер → основной продукт → апсейл

### 2.3 Тренды в автоматизации бизнеса
- **No-code/Low-code платформы:** Make.com, n8n — быстрые интеграции без программирования
- **CRM-центричность:** amoCRM, Bitrix24 как ядро автоматизации
- **Hidden costs awareness:** клиенты знают, что 20-40% от заявленной цены AI-инструментов — скрытые расходы
- **Tool consolidation:** бизнесы хотят сократить количество инструментов, а не добавлять новые

### 2.4 Что лучше всего продается
| Тип проекта | Спрос | Средний чек |
|---|---|---|
| Продающий лендинг под ключ | Высокий | $500-2000 |
| Лендинг + Telegram-бот | Высокий | $1500-4000 |
| Автоворонка (лендинг + бот + CRM) | Очень высокий | $3000-8000+ |
| Интеграция CRM с ботом/сайтом | Средний | $500-2000 |
| E-commerce Telegram бот с Mini App | Растущий | $5000-12000+ |
| Аудит и оптимизация существующей воронки | Средний | $200-800 |

## 3. Боли целевой аудитории

### 3.1 Основные боли (по убыванию важности)

1. **🤯 Разрозненные инструменты («зоопарк систем»)**
   - Бизнес использует 10+ разрозненных платформ
   - Данные дублируются, теряются, тратится время на перенос
   - Сотрудники путаются, контекст-свитчинг убивает продуктивность
   - **Наше решение:** единая экосистема лендинг + бот + CRM

2. **💰 Непонятная окупаемость вложений**
   - Клиенты вложили в сайт/рекламу, но не видят ROI
   - Старая воронка не считается, лиды теряются
   - 33% малого бизнеса не понимают отдачу от digital-инструментов
   - **Наше решение:** сквозная аналитика от клика до продажи

3. **⏱️ Ручная рутина съедает время**
   - Прием заказов, ответы на вопросы, обработка лидов вручную
   - Менеджеры перегружены, отвечают медленно
   - Потеря клиентов из-за медленной реакции
   - **Наше решение:** автоворонки + боты-ассистенты 24/7

4. **📱 Низкая конверсия существующего сайта**
   - Старый сайт не адаптирован под мобайл
   - Долгая загрузка, нет призывов к действию
   - Посетители уходят, не оставив контакт
   - **Наше решение:** современные продающие лендинги

5. **🔒 Безопасность и compliance**
   - Страх утечки данных при внедрении новых инструментов
   - 78% IT-лидеров сталкивались с неожиданными расходами на интеграцию
   - **Наше решение:** прозрачная архитектура, безопасные протоколы

6. **🧠 Нехватка компетенций в команде**
   - Нет специалиста, который может настроить автоматизацию
   - Сотрудники сопротивляются новым инструментам
   - Кривая обучения — 3-6 месяцев просадки продуктивности
   - **Наше решение:** обучение и поддержка на всех этапах

### 3.2 Болевые фразы (можно использовать в текстах)
- «У нас 5 разных сервисов, и они не дружат между собой»
- «Клиенты жалуются, что мы медленно отвечаем»
- «Сделали сайт год назад, а продаж нет»
- «Бот — это круто, но кто его будет делать и поддерживать?»
- «Я не хочу нанимать трех подрядчиков для одного проекта»
- «Мне нужен не просто сайт, а система, которая приносит деньги»

## 4. Портреты клиентов

### Персона A: Владелец микробизнеса / ИП
- **Сфера:** услуги, мелкий ритейл, фитнес, бьюти
- **Бюджет:** 30-80 тыс. ₽
- **Триггер:** «Хочу, чтобы клиенты находили меня и записывались сами»
- **Решение:** лендинг-визитка + Telegram-бот записи
- **Главная ценность:** простота, цена, быстрое запуск

### Персона B: Эксперт / Инфобизнесмен
- **Сфера:** онлайн-курсы, коучинг, консалтинг
- **Бюджет:** 80-200 тыс. ₽
- **Триггер:** «Хочу автоворонку для прогрева и продажи вебинаров»
- **Решение:** многостраничный лендинг + автоворонка + бот-трипваер
- **Главная ценность:** конверсия, автоматизация, аналитика

### Персона C: Владелец малого/среднего бизнеса
- **Сфера:** e-commerce, производство, опт
- **Бюджет:** 150-600 тыс. ₽
- **Триггер:** «Теряю лиды из-за того, что CRM не связана с сайтом и ботом»
- **Решение:** Full funnel: лендинг + интернет-магазин + Telegram-бот + CRM
- **Главная ценность:** интеграции, сквозная аналитика, масштабирование

### Персона D: Стартап / Основатель
- **Сфера:** tech, online-сервисы, SaaS
- **Бюджет:** 200-1000+ тыс. ₽
- **Триггер:** «Нужен MVP с ботом и Mini App для быстрого запуска»
- **Решение:** Telegram Mini App + бот с платежами + API-интеграции
- **Главная ценность:** скорость разработки, технологии, гибкость

## 5. Анализ конкурентов

### 5.1 Типы конкурентов
1. **Фрилансеры-одиночки (70% рынка)**
   - Сильные стороны: низкая цена, личное общение
   - Слабые стороны: нет гарантий, перегружены, нет комплексных решений
   - **Наше преимущество:** системный подход + комплексные решения

2. **Веб-студии (20% рынка)**
   - Сильные стороны: бренд, портфолио, команда
   - Слабые стороны: высокие цены, долгий процесс, шаблонный подход
   - **Наше преимущество:** скорость, гибкость, персональный подход

3. **Конструкторы/маркетплейсы (10% рынка)**
   - Сильные стороны: дешево, быстро, шаблоны
   - Слабые стороны: нет уникальности, ограниченный функционал
   - **Наше преимущество:** кастомные решения под бизнес-задачи

### 5.2 Как выделиться
- **Комплексные пакеты:** лендинг + бот + воронка = одно решение
- **Прозрачность:** фиксированные цены на сайте, сроки, этапы
- **Результаты до/после:** кейсы с реальными метриками
- **Образование:** блог/раздел «Почему это работает» (строим доверие)
- **Демо:** живые примеры работающих ботов и воронок
- **Поддержка:** сопровождение после запуска, а не «сдали и забыли»

## 8. Структура сайта

### 8.1 Главная (лендинг)
```
1. HERO — заголовок + подзаголовок + кнопка CTA
   • «Создаю системы продаж: лендинги, боты, воронки автоматизации»
   
2. ПРОБЛЕМА — блок болей (3-4 боли с которыми приходят)
   • «Узнаёте?» с короткими и ёмкими проблемами
   
3. РЕШЕНИЕ — как мы решаем (услуги)
   • Лендинги, Боты, Воронки, Интеграции
   • Каждая услуга: иконка → описание → цена "от"
   
4. КЕЙСЫ (ПОРТФОЛИО) — 3 лучших проекта
   • Фото/скрин → название → результат в цифрах → «Подробнее»
   
5. ПРОЦЕСС — как мы работаем
   • 4-5 шагов: Брифинг → Дизайн → Разработка → Запуск → Поддержка
   
6. ОТЗЫВЫ — социальное доказательство
   • Реальные отзывы с фото/ссылками
   
7. ПОЧЕМУ Я — преимущества
   • 5-6 пунктов с иконками (опыт, комплексность, поддержка, гарантии)
   
8. FAQ — частые вопросы и ответы
   
9. CTA-СЕКЦИЯ — форма заявки + калькулятор
   • «Рассчитайте стоимость вашего проекта за 2 минуты»
```

### 8.2 Страница портфолио
- Сетка проектов с фильтрацией (лендинги, боты, воронки, интеграции)
- Каждый проект: скриншот/превью → название → короткое описание → результат
- Кнопка «Смотреть кейс» → ведет на детальную страницу

### 8.3 Страница кейса (детально)
```
Структура каждого кейса:
1. Hero: название + главный результат в цифрах
2. Контекст: кто клиент, какая сфера, какие были задачи
3. Проблема: с чем пришел, что не работало
4. Решение: что сделали (текст + скриншоты/видео)
5. Результат: конкретные метрики (до/после)
6. Технологии: какие инструменты использовали
7. Отзыв клиента
8. CTA: «Хочу так же»
```

### 8.4 Дополнительные разделы (по желанию)
- Блог / Полезные статьи
- Страница «Обо мне»
- FAQ по услугам

- ✅ Конкретные цифры и результаты (конверсия, сроки, деньги)
- ✅ Раздел «Почему я» с честными преимуществами

- ✅ Живые демо (тестовый бот, ссылки на проекты)
- ✅ Отзывы с конкретикой и контактами

- Конверсия посетитель → заявка: цель >5%
- Скорость загрузки: <2 сек (desktop), <3 сек (mobile)
- Время на сайте: >2 мин
- Процент дочитывания до CTA: >40%
