## 1. Наше УТП (Уникальное Торговое Предложение)

### Основное УТП (главный заголовок сайта)
> **«Сайт + Бот + Воронка = Система продаж, которая приносит заявки 24/7»**

### Расшифровка
Вместо того чтобы нанимать трех разных подрядчиков (дизайнера, ботодела, CRM-интегратора) — вы получаете единую систему из одних рук. С прозрачной ценой, фиксированными сроками и поддержкой после запуска.

## 4. Коммуникационная стратегия

### 4.1 Тональность
| Параметр | Значение |
|---|---|
| Стиль | Уверенный, экспертный, без воды |
| Формальность | На "ты", профессионально |
| Сложность | Простые слова для сложных вещей |
| Эмоции | Спокойствие, уверенность, доверие |
| Ценности | Результат > Процесс, Польза > Красота |

### 4.2 "Голос" бренда
- Мы говорим о результате, а не о процессе
- Мы объясняем сложное простым языком
- Мы честны: называем сроки и цены
- Мы на стороне клиента: защищаем от переплат за ненужное

### 4.3 Слова-триггеры для текстов
| Использовать | Избегать |
|---|---|
| Заявки, продажи, прибыль | Код, сервер, деплой |
| Автоматизация, система | Фреймворк, библиотека |
| Конверсия, результат | Верстка, адаптив |
| Просто, понятно, надежно | Сложно, мощно, enterprise-grade |
| Бесплатные правки, гарантия | SLA, договорная ответственность |

## 6. Социальное доказательство

### 6.1 Что будем показывать
- **Кейсы с цифрами** — "до" и "после"
- **Отзывы клиентов** — текстовые + видео
- **Сроки работы на рынке** — опыт
- **Технологические партнерства** — если есть
- **Публикации** — статьи, выступления (если будут)

### 6.2 Чего не хватает на рынке (и мы это дадим)
- ❌ Вымышленные цифры в кейсах → ✅ Реальные метрики
- ❌ Отзывы без контактов → ✅ Отзывы с ссылками на клиентов
- ❌ Кейсы без контекста → ✅ Проблема → Решение → Результат
