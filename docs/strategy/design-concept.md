# 📋 Концепция дизайна и оформления

> **Дата:** 2026-06-16 (обновлено)
> **Бренд:** «Цифра» — коллаборация технологий и финансов
> **Статус:** Утверждено после ребрендинга Sol Studio → «Цифра»

---

## 1. Общая концепция

### Стиль: "Технологичный минимализм с теплотой"
- Минимум отвлекающих элементов
- Акцент на контент (кейсы, цифры, результаты)
- Аккуратная микроанимация для жизни
- Доверие через качественный дизайн и прозрачность
- Тёплая, человечная атмосфера (не холодный SaaS)

## 2. Цветовая палитра (УТВЕРЖДЁННАЯ)

| Назначение | HEX | Применение |
|---|---|---|
| Фон основной | #F9F7F4 | Тёплый белый, основной фон секций |
| Фон альтернативный | #F0EDE6 | Серо-бежевый, для контрастных секций |
| Текст основной | #1C1E26 | Глубокий тёмный, заголовки и body |
| Текст muted | #5D5F6E | Приглушённый серый, подписи, мета |
| **Акцент (терракот)** | **#C4604A** | CTA, акценты, hover, ключевые элементы |
| Акцент hover | #A94F3B | Затемнение для терракота |
| Акцент (сланец) | #3A5A6E | Вторичные акценты, финансовая категория |

### Цветовой образ
Тёплая, природная, но современная палитра. Кирпично-терракотовый акцент даёт энергию и тепло. Сланцево-синий добавляет глубину и серьёзность. Не холодный техно-стиль, а «человечные технологии».

## 3. Типографика (УТВЕРЖДЁННАЯ)

### Система из двух шрифтов

| Роль | Шрифт | Начертания | Размеры (desktop) |
|---|---|---|---|
| **Заголовки (h1-h3)** | **Rubik** | Black 900, ExtraBold 800, Bold 700 | h1: 38-56px, h2: 28-40px, h3: 18-22px |
| **Навигация, кнопки** | **Rubik** | Medium 500, SemiBold 600 | 13-16px |
| **Цифры, метрики** | **Rubik** | Black 900 | 34-48px |
| **Текст (body)** | **PT Sans** | Regular 400, Bold 700 | 15-18px |
| **Мелкий текст** | **PT Sans** | Regular 400 | 11-14px |

### Почему такая пара
- **Rubik** — современный гротеск, отлично читается в заголовках, узнаваемый, не AI-шаблонный. Хорошая кириллица в жирных начертаниях.
- **PT Sans** — разработана ParaType специально для кириллицы. Идеальна для длинных текстов: чистая, читаемая, человечная.
- Контраст «гротеск в заголовках + гуманистический в теле» даёт визуальную иерархию без лишних усилий.

### Межстрочные интервалы
- Заголовки: 1.15
- Текст: 1.6
- Маленький текст: 1.4

## 4. Стили компонентов

### Кнопки

| Тип | Стиль |
|---|---|
| Primary CTA | Терракотовый фон, белый текст, скругление 6px, тень при hover |
| Secondary | Прозрачная с тонкой рамкой, при hover — рамка терракотовая |
| Header CTA | Компактная терракотовая, 13px |

### Карточки услуг
- Светлый фон + тонкая рамка (1px)
- Скругление 12px
- Тень + подъём при hover
- Иконка 40×40px на подложке

### Карточки команды (About)
- Две колонки
- Аватар-заглушка с первой буквой (до появления реальных фото)
- Цветные теги компетенций
- Ссылка на Telegram

### Секции
- Чёткое разделение: белый (#F9F7F4) ↔ тёплый серый (#F0EDE6)
- Максимальная ширина контента: 1120px
- Отступы: 56-90px между секциями (desktop), 56px (mobile)

## 5. Анимация и микро-интерактивы

### Рекомендуемые эффекты
- **Цифры-счётчики** — анимация чисел в блоке результатов при скролле
- **Reveal on scroll** — плавное появление элементов (Intersection Observer)
- **Hover на карточках** — лёгкое поднятие + тень (уже реализовано)
- **Аккордеон FAQ** — плавное открытие/закрытие (уже реализовано)
- **Микро-анимация иконок** — лёгкое движение при hover

### Чего избегать
- ❌ Тяжёлой анимации, замедляющей загрузку
- ❌ Мигающих/пульсирующих элементов
- ❌ Автовоспроизведения видео

---

## 6. Иконки и графика

### Стиль иконок
- Кастомные SVG, stroke-style (толщина 1.5px)
- Размер: 28×28px в display, 40×40px контейнер
- Цвет: наследуют акцентный (#C4604A или #3A5A6E в зависимости от категории)
- Lucide-style — линейные, минималистичные

### Изображения (план)
- Скриншоты проектов — единый стиль (тень, скругление 12px)
- Фото команды — реальные, тёплые, в единой тональности
- Логотипы клиентов — grayscale (когда появятся)

---

## 7. Адаптив (Mobile)

### Ключевые точки
- Desktop: >1024px
- Tablet: 900-1024px
- Mobile: <768px

### Mobile-first изменения
- Всё в один столбец на <768px
- Карточки услуг: 3 → 2 → 1 колонка
- Меню в бургер на <900px
- Шрифты: clamp() для плавного изменения
- Отступы: 56px на mobile
