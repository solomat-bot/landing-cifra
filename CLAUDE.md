# Landing Portfolio — Контекст проекта

> **Бренд:** «Цифра» — технологии и финансы для вашего бизнеса
> **Команда:** Ольга Соломатина (технологии) + Евгения Яровая (финансы)
> **Цель:** Привлечение клиентов из микробизнеса (маркетплейсы, онлайн-школы, сервисный бизнес) на финансовое сопровождение и автоматизацию

## Основные направления услуг

### Финансы (Евгения) — ядро
- Управленческий учёт для маркетплейсов (WB, Ozon), онлайн-школ и сервисного бизнеса
- ОПиУ (отчёт о прибылях и убытках), ДДС (движение денежных средств), баланс
- Платежный и кредитный календарь
- ABC-анализ, юнит-экономика, финансовая модель
- Подготовка к кредитованию и инвесторам
- **Цена:** 30 000–75 000 ₽/мес (ежемесячное сопровождение)

### Технологии (Ольга) — усиление + отдельное направление
- Автоматизация сбора данных (связки касса/банк/CRM/TG)
- Сайты и лендинги
- Telegram-боты (запись, заявки, рассылки)
- AI-ассистенты и чат-боты
- API-интеграции
- **Цена:** 5 000–8 000 ₽ (разовые проекты)

### Позиционирование (актуальное с 2026-06-17)
- **Финансы — основной драйвер.** Евгения — 5+ лет опыта, работа с маркетплейсами и онлайн-школами
- **Технологии — усиление.** Автоматизация как новое направление, первые проекты по спеццене
- **Визуальное разделение:** технологический блок — на отдельном фоне со своими границами
- **Имена — только в блоке «Команда».** В продуктах — без имён, фокус на услуге
- AI — помощник, а не замена человеку

## Структура лендинга (17 секций + AI-ассистент)
1. **Header** — лого + «финансы и технологии», навигация (Калькулятор, Тарифы, Кейсы, Сайты и боты, Команда, Статьи)
2. **Hero** — «Хватит гадать, сколько заработали. Наведём порядок в деньгах и процессах.»
3. **Numbers** — 50+ бизнесов, 10+ проектов, 7+ лет, 2 в 1 (команда)
4. **Problems** — 4 боли + ЦА (маркетплейсы, онлайн-школы, сервисный бизнес)
5. **Finance Solution** — учёт + автоматизация (две карточки)
6. **Advantages** — 6 преимуществ
7. **Testimonials** — 3 отзыва от клиентов (социальное доказательство)
8. **Calculator** — 3-шаговая диагностика с ползунком
9. **Pricing Finance** — 3 тарифа (30k/50k/75k) + индивидуальный формат
10. **Tech Services** — 4 карточки, отдельный фон, для тех кому без финансов
11. **Blog** — статьи по финансам и автоматизации
12. **Process** — 4 шага
13. **Case Studies** — 3 демо-кейса (карточки → детальные страницы)
14. **About** — команда (единственное место с именами)
15. **FAQ + Contact** — форма + лид-магнит + прямые контакты
16. **Chat Widget** — плавающая кнопка → диалог с DeepSeek AI-ассистентом
17. **Footer** — лого + «финансы и технологии» + ссылки

## Архитектура проекта
```
landing-portfolio/
├── docs/
│   ├── research/           # Маркетинговые исследования
│   ├── strategy/           # Стратегия и позиционирование
│   └── CHANGELOG.md        # История всех изменений
├── site/
│   ├── src/
│   │   ├── index.html      # Главный лендинг (Цифра) — 17 секций + чат-виджет
│   │   ├── cases.html      # Детальные страницы кейсов (hash-роутинг)
│   │   ├── lead-magnet-checklist.html  # Чек-лист «5 шагов к порядку в финансах»
│   │   ├── og-image.svg    # OG-картинка для соцсетей
│   │   ├── blog.html       # Блог (обновлён под бренд Цифра)
│   │   ├── privacy-policy.html
│   │   ├── blog/           # Статьи блога
│   │   └── tools/          # Инструменты (time-audit-quiz)
│   ├── api/                # Vercel Serverless Functions
│   │   ├── send-message.js # Прокси для Telegram Bot API + выдача чек-листа
│   │   ├── chat.js         # Прокси к DeepSeek API (AI-ассистент)
│   │   └── bot.js          # Webhook-обработчик Telegram-бота продажника @CifraConsultBot
│   ├── assets/             # Статические файлы
│   └── vercel.json         # Конфиг деплоя
├── README.md
└── CLAUDE.md
```
